# GTM 顾问 · 使用场景与提问示例

本文档整理了 GTM 操盘顾问最常见的使用场景，直接复制问题即可提问。

---

## 场景一：新品上市规划

```
# 上市时间线规划
"我们的新品计划在黑五前8周上市，请帮我列出GTM需要在每个阶段完成的核心动作"

# Kick Off 会议准备
"帮我准备一份新品 Kick Off 会议的议程，产品是一款2000元档位的智能手表"

# 4P 准备度检查
"距离上市还有2周，如何快速检查4P各部门的准备度？"
```

---

## 场景二：产品定价与损益

```
# 快速损益测算
"产品BOM成本200元，目标RRP 799元，增值税20%，渠道前向返点15%，
营销费用预算占销额5%，分摊费用固定80元，帮我算净利润"

# 定价策略选择
"我们是新进入欧洲市场的中国品牌，品牌认知度低，
主竞品是索尼和JBL，我该用哪种定价策略？"

# 促销底价测算
"大促期间最低能打到什么价格而不亏本？成本300元，当前街价599"
```

## 场景三：PSI 与库存管理

```
# 库存预警处理
"FBA库存1500台，日均销量20台，大促还有8周，我需要补货吗？怎么算？"

# 高库存处置
"库存DOS已经达到120天，远超健康水位，有哪些清库方式？优先级怎么排？"

# 销售预测偏差
"上个月我预测销量1000台，实际只卖了650台，偏差35%，
供应链要怎么调整这个月的发货计划？"

# SMR会议准备
"帮我梳理销售复盘会（SMR）需要准备哪些数据和交付件"
```

## 场景四：竞品与市场洞察

```
# 竞品分析框架
"我要对标亚马逊德国站上的Anker耳机，帮我列出竞争力分析表应该收集哪些维度"

# 市场洞察报告框架
"帮我生成一个进入泰国智能手环市场的洞察报告大纲"

# 缺少竞品数据
"我找不到竞品的销量数据，有哪些免费和付费的方式可以获取或推算？"

# 量价趋势分析
"过去6个月我品均价从499降到399，但销量只增长了15%，
控比从0.8降到了0.6，这说明什么问题？"
```

## 场景五：产品路标管理

```
# 路标规划

旗舰5999、中端2999、入门999，结合Amazon主要促销节点"

# 老品退市决策
"老品月销量从800台跌到300台，库存还有500台，
我要不要启动退市？退市的触发条件是什么？"

# 新老品切换
"新品即将上市，如何利用老品Listing的热度来养新品？"
```

## 场景七：经营复盘

```
需要涵盖销售、库存、营销ROI和竞争四个维度"

"解释一下GTM经营分析中的同比、环比、控比、盘比，
并用具体例子说明怎么用这四个指标发现问题"
```
